# AI Agent for WhatsApp 💬🤖

Managing customer support on WhatsApp just got easier. **AI Agent for WhatsApp** is an autonomous, AI-powered support agent that runs right on your Mac.

It monitors conversations in real-time and generates rolling AI summaries so you can catch up instantly. But the best part? It can auto-reply to customers using a custom knowledge base you upload!

Built with a strong focus on privacy and safety, it uses on-device **Apple Intelligence** to ensure your data stays secure.

👉 **[Get it on the Mac App Store](https://apps.apple.com/us/app/ai-agent-for-whatsapp/id6759800837)**

## Features
- **Real-Time Monitoring:** Monitors conversations as they happen.
- **Rolling AI Summaries:** Quickly catch up on active chats.
- **Auto-Reply with Knowledge Base:** Automatically answer questions based on your uploaded FAQs.
- **Privacy First:** Powered by on-device Apple Intelligence.
